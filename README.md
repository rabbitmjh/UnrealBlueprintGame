# 📖 프로젝트 소개

**Last Outpost**는 **TPS 기반 웨이브 디펜스 게임**입니다.

플레이어는 이상현상으로 인해 이계와 연결된 지구에서 작전 중 고립된 요원이 되어,
본부의 간접 지원을 받으며 거점을 방어합니다.

웨이브마다 점점 강해지는 몬스터를 상대하며,
드론과 항공 지원을 전략적으로 활용하여 모든 웨이브를 버티고
최종 탈출에 성공하는 것이 목표입니다.

# 🌍 세계관

이상현상으로 인해 지구와 이계가 연결되었습니다.

작전 수행 중 고립된 요원은
본부의 지원만을 받으며 홀로 거점을 방어합니다.

웨이브가 시작될 때마다 본부에서는
적의 규모를 예측하여 브리핑을 제공하며,
모든 웨이브를 막아낸 뒤
탈출 지점까지 돌파하면 임무가 종료됩니다.

```text
        ↓

```

## 몬스터

- 스켈레톤
- 리자드

---

## Object Pool

게임 시작 시 최대 개수만큼 몬스터를 생성합니다.

웨이브마다 필요한 몬스터만 활성화하며,
사망한 몬스터는 비활성화하여 재사용합니다.

시체는 별도의 오브젝트로 교체한 뒤
일정 시간이 지나면 제거합니다.
