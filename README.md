# Last Outpost

> 한정된 탄약과 지원 장비를 관리하며 몬스터 웨이브에서 살아남는 싱글플레이 TPS 웨이브 디펜스 게임

## 프로젝트 소개

**Last Outpost**는 이상현상으로 이계와 연결된 지구를 배경으로 하는 3인칭 슈팅 게임입니다.  
플레이어는 작전 중 고립된 요원이 되어 컨테이너로 구성된 최후의 거점을 방어합니다. 전술 보조 드론과 수류탄, 항공폭격을 활용해 총 5개의 웨이브를 막아내고 끝까지 생존하는 것이 목표입니다.

단순히 모든 적을 처치하는 것뿐 아니라, **탄약과 수류탄, 항공폭격을 적게 사용할수록 더 높은 점수**를 얻도록 설계했습니다.

## 주요 기능

### 전투 및 무기

- 소총, 권총, 샷건, 수류탄 사용
- 무기별 탄창 크기, 예비 탄약, 대미지 및 발사 속도 적용
- 사격, 재장전, 무기 교체 애니메이션
- 샷건 펠릿 확산 처리
- 수류탄 및 항공폭격의 예상 착탄 지점과 폭발 범위 표시
- 지형 피격 흔적과 몬스터 피격 처리

### 플레이어

- 이동, 점프, 달리기, 앉기
- 좌우 몸 기울이기
- 1인칭·3인칭 시점 전환
- 1인칭 조준 시 ADS 확대
- 체력 및 사망 처리

### 몬스터 AI

- 스켈레톤, 드래고노이드, 무장 드래고노이드 3종
- Behavior Tree와 Blackboard 기반 플레이어 탐색·추적·공격
- 공격 범위 판정과 공격 몽타주 Anim Notify 기반 대미지 전달
- 목표 방향을 향한 보간 회전
- 머리 위 월드 스페이스 체력바
- 피격, 사망 애니메이션 및 비활성화 처리
- 스켈레톤의 무기와 라이트는 랜덤

### 웨이브 및 오브젝트 풀

- 총 5개의 웨이브
- 웨이브가 진행될수록 몬스터 종류와 수 증가
- 여러 스폰 지점에서 순차적으로 몬스터 배치
- 몬스터 종류별 20개체, 총 60개체를 미리 생성하는 Object Pool 적용
- 사망한 몬스터를 제거하지 않고 비활성화한 뒤 다음 웨이브에서 재사용

### 전술 보조 드론

- 플레이어를 일정 거리로 따라다니는 비행형 드론
- 주변 몬스터를 탐색해 가장 가까운 대상을 자동 공격
- 목표를 향한 부드러운 회전 및 공격 애니메이션

### 항공폭격

- 폭격 위치와 방향 미리보기
- 마우스 입력으로 실행 또는 취소
- 일정한 간격으로 이어지는 다중 폭발
- 넓은 범위의 몬스터에게 방사형 대미지 적용

### UI 및 게임 흐름

- 시작, 조작 방법, 승리, 패배 화면
- 체력, 탄약, 수류탄, 항공폭격 및 웨이브 정보 표시
- F1 조작 방법 안내
- 모든 웨이브 완료 시 승리, 플레이어 체력이 0이 되면 패배
- 남은 자원을 반영하는 점수 시스템

## 조작법

| `2` | 권총 선택 |

| `Tab` | 1인칭 / 3인칭 시점 전환 |
| `T` | 항공폭격 위치 지정 시작 |
| `F1` | 조작 방법 열기 / 닫기 |

## 게임 밸런스

### 플레이어

| 항목 | 수치 |
| --- | ---: |
| 최대 체력 | 120 |
| 걷기 속도 | 300 |
| 달리기 속도 | 600 |

### 몬스터

| 종류 | 체력 | 공격력 | 공격 범위 | 이동속도 |
| --- | ---: | ---: | ---: | ---: |
| 스켈레톤 | 15 | 5 | 200 | 400 |
| 드래고노이드 | 20 | 10 | 300 | 300 |
| 무장 드래고노이드 | 25 | 15 | 350 | 250 |

### 무기 및 지원 장비

| 장비 | 대미지 | 탄창 | 예비 탄약 / 수량 | 발사 간격 |
| --- | ---: | ---: | ---: | ---: |
| 소총 | 5 | 50 | 300 | 0.1초 |
| 권총 | 3 | 15 | 120 | 0.3초 |
| 샷건 | 펠릿당 8 | 16 | 32 | 1.0초 |
| 수류탄 | 최대 100 | - | 3개 | - |
| 항공폭격 | 폭발당 최대 150 | - | 2회 | 폭발 간격 0.2초 |

수류탄은 최대 450의 폭발 반경을 가지며, 항공폭격은 한 번 실행할 때 총 5회의 연속 폭발을 발생시킵니다. 게임 도중 별도의 탄약 보급 없이 시작할 때 가진 자원을 관리하며 마지막 웨이브까지 생존해야 합니다.

## 핵심 구현 방식

### 몬스터 상태 흐름

```text
Object Pool 대기
→ 웨이브에서 활성화
→ 플레이어 탐색
→ 추적 또는 공격
→ 체력 0
→ 사망 애니메이션
→ 충돌·이동·Tick 비활성화
→ Object Pool 반환
```

### 드론 행동 흐름

```text
플레이어와 거리 확인
→ 거리가 멀면 플레이어 추적
→ 주변 몬스터 탐색
→ 가장 가까운 대상 선택
→ 대상 방향으로 회전
→ 자동 공격
```

## 실행 방법

### 에디터에서 실행

1. Unreal Engine **5.7.4**를 설치합니다.
2. 저장소를 Clone하거나 ZIP 파일로 내려받습니다.
3. 프로젝트 폴더의 `.uproject` 파일을 실행합니다.
4. Unreal Editor에서 `StartMap`을 엽니다.
5. Play 버튼을 눌러 실행합니다.

### 패키징된 버전 실행

배포 페이지에 압축을 해제한 뒤 `LastOutpost.exe`를 실행합니다.
